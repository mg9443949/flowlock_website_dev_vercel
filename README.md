# 🚀 FlowLock

## 📌 About the Project
FlowLock is a **joint collaborative project** focused on building a productivity-enhancing system that helps users stay focused by detecting distractions and enforcing disciplined workflows.

This repository contains the **full-stack implementation** of FlowLock, including frontend, backend, ML modules, and deployment setup.

> ⚡ Note for Recruiters:  
> FlowLock is a **team project**. This repository represents my contributions across multiple parts of the system including development, integration, and deployment.

---

## 💡 What is FlowLock?
FlowLock is a smart productivity platform that combines:
- AI-based distraction detection 🤖  
- Real-time monitoring 📊  
- Multi-device support (Web + Desktop + Mobile) 💻📱  
- Behavioral tracking and analytics 📈  

The goal is to **eliminate distractions and improve deep work efficiency**.

---

## 🧠 Key Features

### 👁️ AI Distraction Detection
- Face-based attention tracking  
- Sound-based distraction monitoring  
- Gesture-based interaction  

### 🔐 Authentication System
- Secure user login and tracking  
- Personalized productivity data  

### 📊 Dashboard & Analytics
- User productivity insights  
- Activity tracking  
- Reports and visualizations  

### 🌐 Cross-Platform Support
- Web app (Next.js)  
- Desktop version  
- Android integration  

### ☁️ Cloud & Deployment
- Hosted on Vercel  
- Backend + database integration (Supabase)
